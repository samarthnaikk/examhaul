# VIT Vellore Past Papers Archive

A comprehensive, organized archive of VIT Vellore's past question papers, with solutions and similar practice questions. This project aims to help students prepare efficiently by providing easy access to previous years' papers, detailed solutions, and curated sets of similar questions for practice.

## Features
- **Organized PDFs:** All past question papers neatly arranged by year, subject, and exam type.
- **Solutions:** Step-by-step solutions for each paper.
- **Similar Questions:** Curated sets of similar questions for additional practice.
- **Search & Filter:** Quickly find papers or questions by subject, year, or topic (planned).
- **Open Source:** Contributions welcome for adding more papers, solutions, or features.


## Getting Started
1. **Clone the repository:**
   ```sh
   git clone https://github.com/yourusername/vit-past-papers.git
   cd vit-past-papers
   ```
2. **Browse the folders:**
   - `papers/` for PDFs
   - `solutions/` for solutions
   - `similar_questions/` for practice
3. **Contribute:**
   - Add new papers, solutions, or similar questions via pull requests.
